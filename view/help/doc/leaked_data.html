Selección de datos y Análisis estadístico básico de las variables involucradas

A partir de los datos importados del Excel y mostrados en forma de tabla en la pestaña Datos filtrados es necesario seleccionar las variables que se desean correlacionar. Para ello, en un recuadro se muestra un inventario de todas las variables importadas. Al seleccionar una variable, se obtiene un resumen estadístico básico de la misma como: cantidad de observaciones, máximo, mínimo, media, mediana, coeficiente de variación (CV), entre otras.
